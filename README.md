# Diabetes Risk Classification using Machine Learning
# Progetto-BI

## Descrizione del progetto
Questo progetto implementa e confronta diversi modelli di **Machine Learning** per la classificazione di pazienti affetti da diabete o a rischio di svilupparlo, a partire da caratteristiche cliniche e demografiche.
Questo progetto rappresenta un **esercizio didattico** svolto nell’ambito di un **corso universitario** e utilizza un dataset di natura **semplice e dimostrativa**, con finalità esclusivamente di studio.

Lo scopo non è sviluppare un sistema clinico reale, ma **analizzare e confrontare diversi modelli di Machine Learning** applicati a un problema di classificazione binaria, mostrando l’importanza delle fasi di preprocessing e di valutazione delle prestazioni.


---

## Dataset
Il dataset è composto da **588 record** e **12 feature**, comprendenti variabili cliniche, demografiche e comportamentali.  
La variabile target indica la **presenza o assenza del diabete**.

---

## Preprocessing
Le principali fasi di preprocessing includono:

- Gestione dei valori mancanti  
- Encoding delle variabili categoriche  
- Rimozione degli outlier  
- Normalizzazione delle feature numeriche  
- Analisi di correlazione e rimozione delle feature ridondanti  
- (Opzionale) Riduzione della dimensionalità con **PCA**

---

## Modelli utilizzati
Sono stati implementati e confrontati i seguenti modelli:

- Decision Tree  
- Random Forest  
- Gradient Boosting  
- K-Nearest Neighbors  
- Support Vector Machine (lineare, sigmoidale, RBF)
